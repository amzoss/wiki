Docker Installation

Before starting with this guide, make sure this installation option makes the most sense for you by reading about the Pros and Cons.

Table of Contents

  1. On Mac
  2. On Windows
  3. On Linux

On Mac

  1. Install Docker by following the instructions on the Docker website

  2. Install Git using the GitHub Desktop App

When you’re done, try following along with the Check for requirements tab to make sure everything is ready to go.

On Windows

  1. Install Docker by following the instructions on the Docker website

  2. Install Git using the GitForWindows toolset

When you’re done, try following along with the Check for requirements tab to make sure everything is ready to go.

*Note: You’ll use the GitBash shell installed along with GitForWindows to check for requirements and run the Wax software.

On Linux

  1. Install Docker by following the instructions on the Docker website

  2. Install Git by following the instructions on the Git website

When you’re done, try following along with the Check for requirements tab to make sure everything is ready to go.
